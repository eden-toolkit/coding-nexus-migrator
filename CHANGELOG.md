# 更新日志

## [1.1.0] - 2025-11-03

### 新增
- 🔍 **进程管理功能** - 新增 `cnm status` 和 `cnm stop` 命令
  - 查看正在运行的迁移进程状态（PID、内存、CPU等）
  - 优雅停止或强制停止迁移进程
  - 支持单进程或多进程批量管理
- 📋 **失败日志追踪** - 详细记录下载和上传失败的依赖路径
  - CODING下载失败日志：`target/failed_downloads.log`
  - Nexus上传失败日志：`target/failed_uploads.log`
  - 包含Maven坐标、文件路径、错误信息和时间戳

### 改进
- 优化内存管理和错误处理
- 增强日志输出和调试信息
